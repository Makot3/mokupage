操作

  • mv 移動元ファイル 移動先ディレクトリ # ファイル移動
  • mv 元ファイル 変更後ファイル # ファイル名変更
  • pwd # カレントディレクトリの確認
  • file ファイル名 # ファイルの種類
  • less ファイル名 # 一画面ずつ
  • cp コピー元 コピー先 # コピー
  • rm ファイル名 # ファイル削除※ゴミ箱経由しない
  • mkdir # ディレクトリ作成

その他操作

  • apt list # パッケージの一覧
  • apt list > packagelist.txt # ってやるといい。多いから
  • apt list --installed # インストールされている一覧
  • apt list --upgradable # 更新可能なパッケージ一覧
  • apt list --installed | wc -l # インストールされているパッケージ数※wc -l は行数を表示
  • apt remove パッケージ名 # アンインストール
  • apt show # パッケージ情報（インストールされていないものもOK）

#検索 grep オプション　パターン　ファイル名 grep http ファイル名　#ファイルからhttpをいう文字列を含む行を抜き出す 　grep echo *.sh #~.shファイルからechoが含まれる行を表示 　grep -i http ファイル名　#大文字小文字にかかわらずhttpで検索

メタキャラ 　. 任意一文字 　* 直前の文字の0回以上繰り返し 　^ 行頭 　$ 行末 　[] カッコ内の1文字 \　直後の1文字はメタキャラではない

#ファイル検索 find 検索ディレクトリ　検索式 　find . -name "*.sh" #ファイル名に.shがつくファイルを検索 　-name 文字列　#ファイル名で検索 　-size 数値　#サイズで検索　+100k:100バイト以上 　-perm アクセス権 　-type ファイルタイプ 　-user ユーザー名 find -perm 600 -exec ls -l {} \; アクセス権が600のファイルを検索して詳細に表示 find -size 0 -type f -exec rm -i {} \; カレント以下の0サイズファイルを消す
