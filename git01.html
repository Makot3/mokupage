Github

初期設定

  • git config --global user.name 'xxx' # 名前登録
  • git config --global user.email 'xxx@xxx.com' # メールアドレス登録 ~/.gitconfigファイルに記録される
  • git init # リポジトリを作りたいディレクトリで。リポジトリの初期化

確認

  • git remote # 設定した名前を表示
  • git branch # ブランチを表示。*付きが現在のブランチ
  • git status # Gitがどのように作業ディレクトリとリポジトリを認識しているか。追跡されていないファイルを表示

ステージングからプッシュ

  • git add # 監視対象にする。ステージング。 git add . でカレント以下全てのファイルをステージする。
  • git commit # コミット。エディタが起動するのでコメントを書く。
  • git push xxx master # Githubへプッシュ。xxxは git remote add xxx git@.... でリポジトリを作成した時の名前

その他

  • git reset HEAD # アンステージ
  • git log # ログを見る。 git log --oneline で一行一履歴表示
  • git diff ID1 ID2 #...
  • git checkout ID File #...
  • git branch ブランチ名 # ブランチ作成
  • git checkout ブランチ名 # ブランチ切り替え
  • git merge ブランチ名 # ...
