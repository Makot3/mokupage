vim

自動インデント設定

cd ~ でホームに移動し、 vim .vimrc で.vimrcというファイルを編集。以下を書く

  • set tabstop=2
  • set shiftwidth=2
  • set expandtab
  • set softtabstop=2
  • set autoindent
  • set smartindent

その他操作

  • 移動は、 hjklでできる
  • i 挿入モード。編集したらEscでノーマルモード。:wqで保存して終了。write, quit
  • / 検索モード。検索したい文字列を入力しエンター。i で挿入モードに入れる。
  • 検索モードからEscでノーマルモードに戻り、n でネクスト検索、Shift+nで逆検索
  • 検索では正規表現も使える
  • コピペ選択は、ノーマルモードで選択したいところまで移動し v
  • コピーは、選択した状態で y (ヤング：引っ張るという意味）
  • 貼り付けは、Shift+p
  • 10Shift+p だと10回貼り付けの意味
  • 切り取りは、選択した状態で d
  • dd で行ごと切り取り

その他操作2

  • アンドゥはノーマルモードで u リドゥは　Ctrl+r
  • Ctrl+z は戻るではなく一時停止。そのアプリがバックグラウンドに移動し端末に戻る。そのアプリを再開するのは fg （フォアグラウンド）
  • 10u 10回アンドゥ、10Cftl+r 10回リドゥ。9999u で最初の状態に戻す
  • ノーマルモードで　Shift+g 最後に移動
  • 置換　ノーマルモードで :%s/元文字列/新しい文字列 この/は[でもいい。:%s は全置換
